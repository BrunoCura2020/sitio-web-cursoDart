¿Qué es Dart?

Dart es un lenguaje de programación de Google, comenzó su desarrollo en el año 2010 y lo presentaron en el 2011. Se puede decir que Dart es de uso general, pero lo que lo hace especial, o yo lo veo así, es que podemos desarrollar aplicaciones moviles tanto para Android como IOS con un mismo código fuente acelerando la productividad.

Variables

Cuando hablamos de variables nos estamos refiriendo a un espacio en memoria donde guardaremos algún tipo de dato. Dicho de otra forma, una variable va a ser quien represente nuestros datos en el programa que estamos programando.

Tener en cuenta:

  • No puede tener un guión(-).
  • No puede comenzar con un número.
  • No se puede usar una palabra reservada del lenguaje para definir variables.
  • Evite usar latras latínas como la (Ñ).
    • <

Ejemplo:

void main(){

var x = 10;

print(x);

}
var es la palabra reservada del lenguaje para crear variables, x es el nombre de la variable, ademas usamos el signo = para asignar un valor a la variable var, adivinen cual es el valor, ¡exacto! 10 y por último lo imprimimos con la otra palabra reservada print.

¿Palabras reservadas?

Cuando decimos palabras reservadas hablamos de aquellas que tienen un significado especial en el lenguaje y no se puede usar como identificadores.

Recomendación

No necesariamente hay que utilizar var, otra forma es escribiendo el tipo de dato, es más, se recomienda especificar de que tipo de datos serán nuestros valores.

Final/Const

Si usas var tendrás la posibilidad de cambiar el valor que ya existe en tu variable, pero si tu intensión no es modificarlo, puedes usar final o const.

Tipos de datos incorporados.

Dart tiene los siguientes tipos datos:

  1. Numbers
  2. String
  3. Booleanos
  4. List
  5. Set
  6. Map

Numbers:

En esta categoría nos encontramos con dos tipos: enteros(int) y decimales(double).

Ejemplo:

void main(){

int nombre_de_variable = 10;

print(nombre_de_variable);

}

void main(){

double nombre_de_variable = 10.2;

print(nombre_de_variable);

}

También tenemos la posibilidad de tener una variable que permita almacenar tanto números enteros como decimales.

void main(){

num nombre_de_variable = 20;

print(nombre_de_variable);

}

void main(){

num nombre_de_variable = 20.5;

print(nombre_de_variable);

}

String:

Éste almacena cadenas de texto; a diferencia del tipo de dato anterior, si o si las cadenas deben ir dentro de unas comillas sin importar si son comillas dobles o simples.

void main(){

String nombre_de_variable1 = "¡Hola!, ésta es una cadena de texto dentro de unas comillas dobles.";

String nombre_de_variable2 = '¡Hola!, ésta es una cadena de texto dentro de unas comillas simples.';

print(nombre_de_variable1);
print(nombre_de_variable2);

}

¡¡Estén atentos a esto!!

Para decirle al programa que la variable va a ser de tipo cadena, siempre pero siempre debemos escribir String con la S en mayúscula.

Dart nos permite usar comillas triples para hacer una cadenas de texto que ocupe más de una línea.

void main(){

String nombre_de_variable = """
¡Hola!, ésta es una cadena de texto
que tiene tres comillas,
así que permite escribir en más de una línea.""";

print(nombre_de_variable);

}

bool:

Cuando hablamos de Booleanos nos estamos refiriendo a dos tipos: true(verdadero) y false(falso). También son muy usados para diferentes tareas como la toma de decisiones dentro de cualquier programa.

List:

La lista es la colección más basica que podemos ver, pero también muy importante; consiste en tener en una misma variable un conjunto de elementos encerrado entre corchete y cada elemento separado por una coma.

void main(){

var nombre_de_variable = [1, 2, 3, 4, 5];

print(nombre_de_variable);

}

No necesariamente tiene que ser de tipo entero. ¡¡Mira los siguientes ejemplos!!.

void main(){

List<String> lista_de_cadenas = ["a", "b", "c"];

print(lista_de_cadenas);

}

void main(){

List<int> lista_de_enteros = [1, 2, 3];

print(lista_de_enteros);

}

void main(){

List<dynamic> lista_dinamica = [1, "a", true];

print(lista_dinamica);

}

Otra cosa interesante que podemos hacer con las listas es almacenar en una lista otra lista, a esto lo podemos llamar "lista de listas".

void main(){

List<dynamic> lista_de_listas = [1, 2, 3, ["a", "b", "c", [ true, false]]];

print(lista_de_listas);

}

Set:

Los Set también es una colección de elementos en una misma variable, pero a diferencia de las listas, en este no se permite elementos repetidos.

void main(){

var no_repite_elementos = <dynamic> {1, 2, 3, 4, 5, 1};

print(no_repite_elementos);

}

El resultado será: {1, 2, 3, 4, 5}, porque como ya te mencioné anteriormente, los elementos no se pueden repetir. Y el editor sabrá cuando estas usando listas o sets porque en las listas se colocan los elementos entre corchetes, mientras que en los set se usan las llaves.

Map:

Los mapas a diferencia de los anteriores, consiste en una colección que tiene clave/valor. El usuario podrá ingresar al valor si imprimé la clave.

void main(){

Map:<String, dynamic> miMapa = {"Dart":"Flutter"; "nacimiento":2011};

print(miMapa);

print(miMapa["Dart"]);

}

Quiero que se den cuenta de algo, como ven he colocado dos tipos de datos, y lo hice porque el primero(String) representa el tipo de dato de la clave y el segundo va a representar el tipo de dato de los valores. Con eso quiero decir que, la clave si o si tiene que ser de tipo cadena y el valor puede ser de cualquier tipo, osea es dinámico. Y por último se puede ver dos print, aquél que tiene un corchete será quien muestre el valor de la clave y el otro mostrará el mapa completo.

Les aclaró algo en cuánto a las colecciones de elementos, yo solamente les di una pequeña introducción, si investigan más verán que es todo un mundo, encontrán lo que son los métodos por ejemplo, con los métodos podrás agregar elementos, eliminarlos, cambiar de posición, saber en que posición se encuentra, y un gran etc..

== != > < >= <=

¡¡Atención!!

No confundan el = con el ==, el primero nos sirve para decirle que le estamos asignando un valor, y el segundo tiene la función de hacer comparaciones. Por ejemplo:
int a = 10;
int b = 20;
print(a == b);
false: porque a(10) no es igual a b(20);
